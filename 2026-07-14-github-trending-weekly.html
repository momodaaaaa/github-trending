本周 GitHub 热门项目推荐

这周的主题很明确：AI Agent 工具生态正在快速成型。 18 个项目里有 8 个直接跟 Agent 工具链相关，从 Office 自动化到多 Agent 调度、从代码审查到网页操作，覆盖面非常广。 同时安全测试和视频分析也在被 AI 改造，而一批 AI 求职项目冒出来——找工作这件事也在被 AI 改变。

AI 求职 & 个人自动化

01
ai-job-search
MadsLorentzen/ai-job-searchTypeScript本周 +15,420 ⭐总 ⭐22.2k
跑在本地的 AI 求职框架，基于 Claude Code 运行。它能自动扫招聘信息、根据你的背景匹配合适的岗位，然后帮你生成定制化的简历和求职信。配好你的信息之后，投简历、跟踪进度这些重复劳动就能交给它处理。在找工作的话，相当于把海投阶段外包给了 AI，自己专注在匹配度更高的机会上。全程跑在你本地，数据不上传任何第三方。单周冲到 1.5 万星，说明确实戳中了很多人的实际需求。
02

办公和自动化

04

AI Agent 生态
